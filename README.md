# System Capacity & Care Load Analytics for Unaccompanied Children

This project analyzes healthcare system capacity using Python, Pandas, Matplotlib, and Streamlit.

## Features
- Data Cleaning
- Exploratory Data Analysis
- KPI Summary
- Time-Series Visualizations
- Streamlit Dashboard

## Technologies
- Python
- Pandas
- Matplotlib
- Streamlit
